# 백테스팅 및 검증 계획서 (Backtesting & Validation)

## 1. 검증 목표
실제 자산을 투입하기 전, 과거 데이터를 통해 '변동성 조절형 평균 회귀' 전략의 수익성과 안정성을 통계적으로 증명합니다.

## 2. 백테스팅 단계
### 1단계: 과거 데이터 시뮬레이션
- **대상 기간:** 최근 1년 (상승장, 하락장, 횡보장 포함)
- **데이터 단위:** 15분봉 또는 1시간봉 기준
- **검증 지표:** - 승률 (Win Rate)
    - 손익비 (Profit/Loss Ratio)
    - 최대 낙폭 (MDD: Maximum Drawdown)

### 2단계: 가상 매매 (Paper Trading)
- **방법:** 실제 업비트 시세 데이터를 실시간으로 받아오되, 주문만 가상으로 처리.
- **기간:** 최소 1주일 이상
- **목표:** 실제 네트워크 지연 시간(Latency)과 슬리피지가 수익률에 미치는 영향 확인.

## 3. 성공 기준 (Exit Criteria)
- 백테스팅 결과 **MDD가 10% 이내**로 유지될 것.
- 가상 매매 시나리오에서 **수수료를 제외하고도 양(+)의 수익률**을 기록할 것.
